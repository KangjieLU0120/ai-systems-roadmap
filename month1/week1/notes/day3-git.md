# Day 3 - Git Basics

## Commands learned today

- git init
- git status
- git add
- git commit
- git log
- git diff

## Concepts

### Working directory

The working directory is where I directly edit files.

### Staging area

The staging area is where I prepare changes before committing them.

### Commit

A commit is a saved snapshot of selected changes in the project.

### Why commit frequently

Frequent commits make changes easier to review, understand, roll back, and share.
